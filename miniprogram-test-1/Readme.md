小程序有两个tabbar，第一个是首页，设想是等后续服务端完成后该页面向服务端请求任务列表，点击某个任务后可以跳转到该任务详情页。首页上方有一个搜索框，点击该搜索框跳转到搜索页。
第二个tabbar是“我的”，后续讨论后可以加入用户信息、已完成任务、收入等信息。
